TickerForge

Privacy Policy

Last updated: February 2025

01

Introduction

TickerForge is a browser extension for monitoring the cryptocurrency market. This Privacy Policy explains what data the extension collects, how it is used, and how it is stored. We are committed to protecting your privacy and being transparent about our data practices.

02

Data Collection

TickerForge does not collect, transmit, or store any personal data on external servers.

The extension does not:

  • Require account creation or registration
  • Collect names, email addresses, or any personal identifiers
  • Use analytics, telemetry, or tracking services
  • Use cookies or fingerprinting techniques
  • Access browsing history, bookmarks, or page content
  • Inject scripts into web pages
  • Transmit any user data to third parties
03

Locally Stored Data

All user data is stored exclusively on your device using the Chrome storage.local API. This includes:

  • User preferences — selected theme, language, display currency, keyboard shortcut, feed refresh interval, and provider preference
  • Watchlist — list of cryptocurrency assets you choose to monitor
  • Cached API responses — temporarily stored market data, news headlines, and chart data to reduce network requests and improve performance

This data never leaves your browser and is not accessible to any external service or server.

04

Network Requests

TickerForge makes requests to the following public APIs solely to retrieve cryptocurrency market data:

  • api.coingecko.com — Market data, prices, charts
  • api.coincap.io — Fallback market data
  • min-api.cryptocompare.com — Cryptocurrency news
  • api.alternative.me — Fear & Greed Index

These requests contain no personal information — only standard HTTP headers and query parameters required to fetch public market data (e.g., coin ID, time range, currency). No API keys, authentication tokens, or user identifiers are sent with these requests.

05

Permissions

The extension requests the following browser permissions:

  • storage — To save your preferences, watchlist, and cached data locally on your device
  • alarms — To schedule periodic background updates of market data, news, and watchlist prices
  • Host access to API domains — To fetch cryptocurrency data from the public APIs listed in Section 4

The extension does not request access to your tabs, browsing activity, active page content, or any other sensitive browser data.

06

Third-Party Services

TickerForge uses third-party APIs exclusively to retrieve publicly available cryptocurrency data. We do not control these services and are not responsible for their privacy practices. No user data is shared with these providers — requests consist only of standard market data queries.

07

Children's Privacy

TickerForge does not knowingly collect any personal information from anyone, including children under the age of 13. The extension is a market data display tool and does not facilitate financial transactions, account creation, or any form of data submission.

08

Data Retention and Deletion

Since all data is stored locally on your device:

  • To clear cached data — use the extension's built-in settings or clear your browser's extension storage
  • To delete all data — uninstalling the extension automatically removes all associated local data

No data persists on any external server because no data is ever sent to one.

09

Security

TickerForge is built with security in mind:

  • All network requests are made over HTTPS
  • No remote code execution — all logic is bundled within the extension
  • No use of eval(), inline scripts, or dynamic code loading
  • Content Security Policy (CSP) is enforced per Manifest V3 requirements
  • No sensitive credentials are stored or transmitted
10

Changes to This Policy

We may update this Privacy Policy from time to time. Any changes will be reflected in the extension update. Continued use of the extension after changes constitutes acceptance of the updated policy.
